一、两种成卦法的本质定位
方法   名称   定位   输出   功能
A法   河洛真数正统成卦法（天地余数合成重卦）   先天之“体”   一个六爻重卦（如地天泰䷊）   判断一生大运格局、贵贱层次、顺逆趋势

B法   年干起卦 + 动爻变卦法（纳甲命盘模型）   后天之“用”   一个带六亲、世应、元堂的六爻卦   分析父母、配偶、子女、财运、灾厄等具体事项

✅ 核心思想：  
A法定“命格高度”，B法定“人生细节”。  
就像一栋楼：A法看是别墅还是危房，B法看厨房在哪、水管是否漏水。

二、结合逻辑：三步走策略

第一步：用 A 法定“命卦”（体）
- 通过四柱奇偶数分离 → 天数/地数 → 余数 → 上下卦 → 合成先天命卦。
- 此卦代表命运底层代码，不随流年改变。

第二步：用 B 法建“命盘”（用）
- 年干定本命单卦 → 四柱总和定动爻 → 成变卦 → 装六亲、定元堂。
- 此盘用于具体人事推演，可结合化工、流年动态分析。

第三步：体用互参，综合断事
- 若命卦吉 + 命盘吉 → 大富大贵；
- 若命卦凶 + 命盘吉 → 有才无运，或富贵不长久；
- 若命卦吉 + 命盘凶 → 怀才不遇，需待时而动；
- 若命卦凶 + 命盘凶 → 多灾多难，宜守不宜攻。

📌 关键原则：  
命卦管“能不能成”，命盘管“怎么成、何时成”。

三、完整案例演示

命造：  
男性，出生于 1985年3月20日巳时  
公历：1985年3月20日 9:30（东八区）  
农历：乙丑年二月廿九日 巳时  
四柱：乙丑 己卯 丁未 乙巳

▶ 第一步：求数法（统一输入）

按河图/洛书取数：
柱   天干   河图数   地支   洛书数
年   乙     2（偶）  丑     2（偶）

月   己     6（偶）  卯     4（偶）

日   丁     4（偶）  未     8（偶）

时   乙     2（偶）  巳     6（偶）

⚠️ 注意：此命四柱全为偶数！无奇数。

- 天数和（奇数） = 0  
- 地数和（偶数） = 2+2+6+4+4+8+2+6 = 34

▶ 第二步：A法 —— 正统成卦（定体）

1. 天数余 = (0 − 25) → 负数，视为 0 → 取 0 → 按古法，0 视为 10 → 个位 0 → 实际多取“寄宫”或参考年干。  
   - 更稳妥做法：因无天数，上卦取年干对应卦 → 乙 → 坤
   
2. 地数余 = 34 − 30 = 4 → 对应 巽卦（四巽）

3. 性别与阴阳：
   - 年干乙 → 阴
   - 男性 → 阴男
   - 口诀：“阴男阳女逆” → 上卦=地卦，下卦=天卦

   但此处天数为0，我们采用修正规则：  
   > 若天数为0，则以年干卦为下卦，地数余卦为上卦。

   → 下卦：乙 → 坤  
   → 上卦：余4 → 巽

4. 命卦 = 风地观䷓（☴上 ☷下）

📌 观卦象义：  
“盥而不荐，有孚颙若。”——主观察、等待、以静制动。  
此命宜幕后策划、学术研究、宗教文化，不宜强求名利。早年平淡，中年后声望渐起。

✅ A法结论：先天格局为文人贵格，非暴发之命，需厚积薄发。

▶ 第三步：B法 —— 纳甲命盘（定用）

1. 本命卦：年干乙 → 坤卦（☷）

2. 四柱总和 = 34  
   → 34 ÷ 6 = 5 余 4 → 四爻动

3. 变卦：坤卦四爻动 → 上卦震（☳），下卦坤（☷） → 雷地豫䷏

4. 装六亲（以日干“丁火”为“我”）：
   - 坤土爻 → 火生土 → 子孙爻
   - 四爻子孙动 → 主靠技艺、创意、子女成名

5. 元堂定位：
   - 阴男，生于未日（地支序号=8）
   - 从上爻（六爻）逆数8位：6→5→4→3→2→1→6→5 → 停在五爻
   - 元堂在五爻（君位）→ 主中年掌权、有名望

6. 结合化工图：
   - 生于卯月（木旺）→ 木克土 → 子孙爻受克
   - 但日支未土、时支巳火 → 有根有生 → 克中有救

✅ B法结论：
- 子女缘有但晚得，或子女体弱；
- 本人有艺术/技术才华（子孙动），但需防因创意项目破财（木克土）；
- 中年（元堂在五爻）事业登顶。

▶ 第四步：体用结合，综合断事

四、总结：为何必须结合？

- 只用A法：知道他是“观卦之人”，但不知他几岁结婚、父母是否健在；
- 只用B法：知道他“子孙动”，但不知这是天赋使命还是短暂机遇；
- 二者结合：既知其命格高度（观卦之清贵），又明其实现路径（豫卦之悦乐），方能“知命造运”。
