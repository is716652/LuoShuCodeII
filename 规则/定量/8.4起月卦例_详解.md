# 起月卦例与六约卦详解

## 一、 核心概念

“起月卦”是河洛真数中用于推算**流月运势**的核心方法。它结合了**元堂爻**（先天卦的核心爻位）与**流月干支**，通过特定的法则确定每个月的**月卦**（卦象）和**主爻**（动爻），并结合五行生克来判断吉凶。

此法又称“六约卦”，因为它将一年分为六个阶段（每两个月为一个阶段，或视每个月为一个独立节点），细致描绘了一年之内的气运起伏。

## 二、 推算步骤

推算流月运势主要包含三个步骤：**定月卦**、**移爻位**、**判吉凶**。

### 1. 定月卦（由月支定卦）

月卦的卦象由该月的**地支**决定。根据河洛真数（及后天八卦方位理论），十二地支对应八卦的映射关系如下：

| 月支 | 对应卦象 | 五行 | 方位/解释 |
| :--- | :--- | :--- | :--- |
| **子 (鼠)** | **坎** (水) | 水 | 正北，水旺之地 |
| **丑 (牛)** | **艮** (山) | 土 | 东北，土金相生 |
| **寅 (虎)** | **震** (雷) | 木 | 东北偏东，木气始生 |
| **卯 (兔)** | **巽** (风) | 木 | 正东，木气正旺 (此处取巽代震之位或重排) |
| **辰 (龙)** | **巽** (风) | 木/土 | 东南，木余气 (此处归巽) |
| **巳 (蛇)** | **离** (火) | 火 | 东南偏南，火气始生 |
| **午 (马)** | **离** (火) | 火 | 正南，火气正旺 |
| **未 (羊)** | **坤** (地) | 土 | 西南，土气旺 |
| **申 (猴)** | **乾** (天) | 金 | 西南偏西，金气始生 |
| **酉 (鸡)** | **兑** (泽) | 金 | 正西，金气正旺 |
| **戌 (狗)** | **乾** (天) | 金/土 | 西北，金余气 (此处归乾) |
| **亥 (猪)** | **坎** (水) | 水 | 西北偏北，水气始生 |

### 2. 移爻位（月移一爻）

流月的**主爻位**（即该月运势的触发点）是在**先天元堂爻**的基础上，随着月份的推移而逐月上移。

*   **起点**：正月（一月）的主爻位 = **先天元堂爻位**。
*   **规律**：每过一个月，爻位向上移动一位。
*   **循环**：如果移到了上爻（第6爻），下一个月则回到初爻（第1爻）。

**计算公式**：
```text
流月主爻位 = (先天元堂爻位 + 月份 - 2) % 6 + 1
(注：月份为农历月份，1-12)
```

**示例**：
假设某人先天元堂爻为 **三爻**（第3爻）：
*   **正月**：主爻为 **三爻**。
*   **二月**：主爻为 **四爻**。
*   **三月**：主爻为 **五爻**。
*   **四月**：主爻为 **上爻**。
*   **五月**：主爻回到 **初爻**。
*   ...以此类推。

### 3. 判吉凶（五行生克）

吉凶的判断主要依据**月支五行**与**日元（日主）五行**的关系。

*   **吉（利）**：
    *   **月生身**：月支五行 **生** 日元五行（如日元属木，月支属水，水生木）。此为印绶之助，主贵人扶持、舒适、得力。
    *   **月同身**：月支五行 **同** 日元五行（如日元属木，月支属木）。此为比劫之助，主朋友帮扶、自信、合伙。

*   **凶（不利）**：
    *   **月克身**：月支五行 **克** 日元五行（如日元属木，月支属金）。主压力、官非、阻滞。
    *   **身克月**：日元五行 **克** 月支五行（如日元属木，月支属土）。虽为财星，但身弱者恐不胜财，主劳累、耗损。
    *   **身生月**：日元五行 **生** 月支五行（如日元属木，月支属火）。此为食伤泄秀，虽有才华发挥，但主泄气、操劳。

> **特别说明**：目前的简化逻辑主要看“生我”和“同我”为吉，其余情况（克、泄、耗）偏向于凶或平，具体需结合命局强弱细断。

## 四、 总结

“起月卦”通过将时间（流月）的空间化（卦象）和动态化（爻位移动），提供了一套精细的运势扫描机制。
1.  **卦象**揭示了当月的大环境气场（如坎卦主险、离卦主明）。
2.  **爻位**指出了当月的关键作用点（初爻为潜、五爻为尊等）。
3.  **五行**定夺了当月的最终得失（吉凶）。

掌握此法，便可对一年十二个月的运程了然于胸。
