# 河洛真数：定元堂式详解

**“定元堂式”**是《河洛真数》中用于锁定卦象核心（元堂爻）的计算法则。此步骤紧接在“八卦相荡”之后——先定卦，后定爻。

---

## 一、 什么是“元堂”？

如果说“六十四卦”描绘了一个人的宏观命运蓝图（如“风山渐”卦），那么**“元堂爻”**就是这幅蓝图中**属于你自己的那个具体位置**。

*   **地位**：它是卦的“心”，是吉凶的聚焦点。
*   **作用**：全卦的吉凶，最终都要看元堂爻是否得位、是否得时、是否有应援。
    *   **吉**：元堂爻得位（如君子居高位），主富贵、长寿。
    *   **凶**：元堂爻失位（如小人窃权或君子受困），主贫贱、灾咎。

---

## 二、 定元堂的总纲（基础公式）

定元堂的核心依据是**出生时辰**（十二地支）。

### 1. 时辰分阴阳
*   **阳时（子至巳）**：子、丑、寅、卯、辰、巳。
    *   **法则**：**阳时报阳**。即出生在阳时的人，元堂爻优先落在卦中的**阳爻**上。
*   **阴时（午至亥）**：午、未、申、酉、戌、亥。
    *   **法则**：**阴时报阴**。即出生在阴时的人，元堂爻优先落在卦中的**阴爻**上。

### 2. 数位方向
*   绝大多数情况下，是**“自下而上”**数。
*   即从初爻开始往上数，数到上爻若还没数完，再回到下面继续数（往复循环）。

---

## 三、 杂卦（混卦）定式

**杂卦**是指卦中既有阴爻又有阳爻的卦（除乾、坤以外的62卦）。

### (一) 阳时生人（找阳爻）

根据卦中**阳爻的数量**来分配时辰：

| 卦中阳爻数 | 时辰分配法则（自下而上） | 举例（以子丑寅卯辰巳为序） |
| :--- | :--- | :--- |
| **1个阳爻** | 该阳爻统管所有阳时（前2个时辰） | **子、丑**都在这一根阳爻上。（寅卯辰巳借阴位，见特殊规则） |
| **2个阳爻** | 两阳爻循环数两次 | 下阳爻管**子、寅**；上阳爻管**丑、卯**。（辰巳借阴位） |
| **3个阳爻** | 三阳爻循环数两次（刚好管满6个阳时） | 初阳(子)、二阳(丑)、三阳(寅)、回初(卯)、回二(辰)、回三(巳)。 |
| **4个阳爻** | 四阳爻各管一时 | 自下而上：**子、丑、寅、卯**。（辰巳借阴位） |
| **5个阳爻** | 五阳爻各管一时 | 自下而上：**子、丑、寅、卯、辰**。（巳借阴位） |

*   **注**：若阳时生人，但卦中阳爻位置不够（如1阳爻只管子丑），后面时辰（寅卯辰巳）怎么排？
    *   **借位法则**：阳位排满后，借用阴爻的位置，依然是自下而上排。

### (二) 阴时生人（找阴爻）

根据卦中**阴爻的数量**来分配时辰：

| 卦中阴爻数 | 时辰分配法则（自下而上） | 举例（以午未申酉戌亥为序） |
| :--- | :--- | :--- |
| **1个阴爻** | 该阴爻统管所有阴时（前2个时辰） | **午、未**都在这一根阴爻上。 |
| **2个阴爻** | 两阴爻循环数两次 | 下阴爻管**午、申**；上阴爻管**未、酉**。 |
| **3个阴爻** | 三阴爻循环数两次（刚好管满6个阴时） | 初阴(午)、二阴(未)、三阴(申)、回初(酉)、回二(戌)、回三(亥)。 |
| **4个阴爻** | 四阴爻各管一时 | 自下而上：**午、未、申、酉**。 |
| **5个阴爻** | 五阴爻各管一时 | 自下而上：**午、未、申、酉、戌**。 |

## 四、 纯卦（乾坤）定式

**乾卦（六阳）**与**坤卦（六阴）**因为没有异性爻可借，取法最为复杂，涉及“节气”与“顺逆”。

### 1. 乾卦（全阳）

| 命造 | 生时 | 节气 | 定式 |
| :--- | :--- | :--- | :--- |
| **男** | 阳时 | 任意 | **下卦**（初二三爻）循环数：子寅辰在初二三，丑卯巳在初二三。（不出下卦） |
| **男** | 阴时 | 任意 | **上卦**（四五上爻）循环数：午申戌在四五上，未酉亥在四五上。（不出上卦） |
| **女** | 阳时 | **冬至后** | **上卦**，**自上而下**逆数。（上→五→四...） |
| **女** | 阴时 | **冬至后** | **下卦**，**自上而下**逆数。（三→二→初...） |
| **女** | 阳时 | **夏至后** | 同男命阳时（下卦顺数）。 |
| **女** | 阴时 | **夏至后** | 同男命阴时（上卦顺数）。 |

### 2. 坤卦（全阴）

| 命造 | 生时 | 节气 | 定式 |
| :--- | :--- | :--- | :--- |
| **女** | 阳时 | 任意 | **下卦**，自下而上顺数。（同乾卦男命阳时逻辑） |
| **女** | 阴时 | 任意 | **上卦**，自下而上顺数。（同乾卦男命阴时逻辑） |
| **男** | 阳时 | **夏至后** | **上卦**，**自上而下**逆数。 |
| **男** | 阴时 | **夏至后** | **下卦**，**自上而下**逆数。 |
| **男** | 阳时 | **冬至后** | 同女命阳时（下卦顺数）。 |
| **男** | 阴时 | **冬至后** | 同女命阴时（上卦顺数）。 |

> **纯卦口诀**：
> 乾男坤女为正格，阳时在下阴在上。
> 乾女坤男为变格，冬夏二至分顺逆。
> 顺则同正逆则反，上下颠倒数法易。

## 五、 案例演示

**例：** 接前文“风山渐”卦（上巽下艮）。
*   **卦象**：
    *   上爻 (阳)
    *   五爻 (阳)
    *   四爻 (阴)
    *   三爻 (阳)
    *   二爻 (阴)
    *   初爻 (阴)
    *   **统计**：3个阳爻（三、五、上），3个阴爻（初、二、四）。

**假设1：阳命男，生于“寅时”（阳时）。**
*   **法则**：3个阳爻，管6个阳时。
*   **排序**：
    *   子时 -> 下阳爻 (三爻)
    *   丑时 -> 中阳爻 (五爻)
    *   **寅时 -> 上阳爻 (上爻)**
*   **结论**：元堂爻在**上九**。

**假设2：阳命男，生于“午时”（阴时）。**
*   **法则**：3个阴爻，管6个阴时。
*   **排序**：
    *   **午时 -> 下阴爻 (初爻)**
*   **结论**：元堂爻在**初六**。

通过“定元堂式”，我们就能在茫茫卦象中，精准找到代表命主的那一个点。
