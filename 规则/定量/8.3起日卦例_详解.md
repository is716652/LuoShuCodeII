# 起日卦例与流日运势详解

## 一、 核心概念

“起日卦”是河洛真数中用于推算**流日（每日）运势**的方法。与流月卦类似，它通过**流日干支**来确定当日的**日卦**（卦象）和**动爻**，从而判断当日的气场吉凶。

此法可用于指导日常行事，如出行、谈判、交易等。

## 二、 推算步骤

推算流日运势主要包含三个步骤：**定日卦**、**定动爻**、**判吉凶**。

### 1. 定日卦（由日支定卦）

日卦的卦象由当日的**地支**决定。其映射关系与流月卦完全一致（基于后天八卦方位或纳甲）：

| 日支 | 对应卦象 | 五行 | 方位/解释 |
| :--- | :--- | :--- | :--- |
| **子 (鼠)** | **坎** (水) | 水 | 正北 |
| **丑 (牛)** | **艮** (山) | 土 | 东北 |
| **寅 (虎)** | **震** (雷) | 木 | 东北 |
| **卯 (兔)** | **巽** (风) | 木 | 正东 |
| **辰 (龙)** | **巽** (风) | 木/土 | 东南 |
| **巳 (蛇)** | **离** (火) | 火 | 东南 |
| **午 (马)** | **离** (火) | 火 | 正南 |
| **未 (羊)** | **坤** (地) | 土 | 西南 |
| **申 (猴)** | **乾** (天) | 金 | 西南 |
| **酉 (鸡)** | **兑** (泽) | 金 | 正西 |
| **戌 (狗)** | **乾** (天) | 金/土 | 西北 |
| **亥 (猪)** | **坎** (水) | 水 | 西北 |

### 2. 定动爻（日支定爻）

流日的**动爻位**（即当日的关键爻位）直接由**日支的序数**决定，呈周期性循环（1-6）。

*   **计算公式**：
    ```text
    流日动爻位 = (日支序数) % 6 + 1
    ```
    *(注：日支序数 子=0, 丑=1, ..., 亥=11)*

*   **对照表**：

| 日支 | 序数 | 计算 | **动爻位** |
| :--- | :--- | :--- | :--- |
| **子、午** | 0, 6 | 0%6+1, 6%6+1 | **初爻** |
| **丑、未** | 1, 7 | 1%6+1, 7%6+1 | **二爻** |
| **寅、申** | 2, 8 | 2%6+1, 8%6+1 | **三爻** |
| **卯、酉** | 3, 9 | 3%6+1, 9%6+1 | **四爻** |
| **辰、戌** | 4, 10 | 4%6+1, 10%6+1 | **五爻** |
| **巳、亥** | 5, 11 | 5%6+1, 11%6+1 | **上爻** |

### 3. 判吉凶（五行生克）

吉凶的判断依据**日支五行**与**日元（日主）五行**的关系。

*   **吉（利）**：
    *   **日生身**（印）：日支五行 **生** 日元五行。
    *   **日同身**（比）：日支五行 **同** 日元五行。
    *   **宜**：出行、签约、开业、求财。
    *   **忌**：诉讼、冲突。

*   **凶（不利）**：
    *   **日克身**（官杀）：日支五行 **克** 日元五行。
    *   **宜**：祈福、休养、读书。
    *   **忌**：出行、签约、开业、大事。

*   **平（中平）**：
    *   **身克日**（财）、**身生日**（食伤）。
    *   **宜**：日常事务、交友、学习。
    *   **忌**：重大决定。

## 三、 实战举例

**假设条件**：
*   **日元**：庚金（五行属**金**）。
*   **日期**：某年某月 **丙寅日**。

**推算过程**：
1.  **定日卦**：日支为 **寅** -> 对应 **震卦**（雷）。
2.  **定动爻**：寅对应序数2 -> (2)%6 + 1 = **三爻**。
3.  **判吉凶**：
    *   日支（寅）五行为 **木**。
    *   日元（庚）五行为 **金**。
    *   关系：**金克木**（我克者为财）。
    *   结论：**平**（或偏吉，视身强弱而定，但在简化规则中归为平/耗）。

**当日运势**：
*   **卦象**：震为雷（主动、响动）。
*   **关键点**：三爻（处于下卦之极，恐有躁动）。
*   **综合**：运势平稳，虽有求财之机（我克为财），但需防劳累（耗气）。

## 四、 总结

起日卦法简单明了，无需复杂的元堂推演，直接以“日支”为核心，快速定位当日的**环境气场（卦）**与**行动节奏（爻）**，是日常趋吉避凶的实用小工具。
