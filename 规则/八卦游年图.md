## 核心概念解析

### 什么是"游年"？

"游年"即"游走的能量"。在易学（特别是八宅风水与河洛真数）中，任何两个卦象相遇，都会产生一种特定的**感应气场**。

*   **主体（本命/坐山）**：通常代表"我"或"根本"。
*   **客体（方位/大运）**：通常代表"环境"或"遭遇"。
*   **游年星**：两者碰撞后产生的吉凶结果。

### 能量的八种形态（八星）

古人将这种碰撞结果分为八类，赋予了人格化的名字（八星），并对应五行属性与吉凶。

| 游年星 | 五行 | 吉凶 | 核心含义 | 记忆口诀 |
| :--- | :--- | :--- | :--- | :--- |
| **生气** | 木 | **大吉** | 生机勃勃，事业财运双旺，精力充沛。 | **上爻变** |
| **天医** | 土 | **中吉** | 祛病延年，贵人相助，身体健康。 | **中下变** |
| **延年** | 金 | **小吉** | 感情和睦，人际和谐，家庭圆满。 | **全变** |
| **伏位** | 木 | **小吉** | 平稳安定，蓄势待发，原本状态。 | **全不变** |
| **绝命** | 金 | **大凶** | 破败损丁，意外灾祸，顽疾难愈。 | **中爻变** |
| **五鬼** | 火 | **次凶** | 口舌官非，火灾失窃，小人作祟。 | **上中变** |

## 变爻法则：吉凶推导的底层逻辑

这是理解游年图的**金钥匙**。不需要死记硬背口诀，只要看两个卦的**三爻（初、二、三）**是如何变化的，就能瞬间推出结果。

### 变爻对照表（通用公式）

| 变化情况 | 对应星名 | 记忆技巧 |
| :--- | :--- | :--- |
| **上爻变** | **生气** | 只有最上面的线不一样，想往上长 -> 生气 |
| **中爻变** | **绝命** | 只有中间的心变了，心死则绝 -> 绝命 |
| **下爻变** | **祸害** | 只有根基动摇了，祸起萧墙 -> 祸害 |
| **上+中 变** | **五鬼** | 上面中间都乱了，群魔乱舞 -> 五鬼 |
| **上+下 变** | **六煞** | 上下不通，中间受困，水流乱窜 -> 六煞 |
| **中+下 变** | **天医** | 根基和内心都变好了（去病），只留头脑清醒 -> 天医 |
| **三爻全变** | **延年** | 彻底翻转，阴阳互补，如夫妻互补 -> 延年 |
| **三爻不变** | **伏位** | 完全一样，重叠不动 -> 伏位 |

## 实战举例（详细推演）

为了让你真正学会，我们不只看乾卦，我们换几个卦来推演。

### 【案例一】 坎卦（☵）为命主，遇到 巽卦（☴）

*   **第一步：画出卦象**
    *   **坎卦（主）**：☵ （阴、阳、阴）
        *   上爻：阴 (- -)
        *   中爻：阳 (———)
        *   下爻：阴 (- -)
    *   **巽卦（客）**：☴ （阳、阳、阴）
        *   上爻：阳 (———)
        *   中爻：阳 (———)
        *   下爻：阴 (- -)

*   **第二步：逐爻对比**
    *   **上爻**：坎(阴) vs 巽(阳) -> **不同（变）**
    *   **中爻**：坎(阳) vs 巽(阳) -> 相同（不变）
    *   **下爻**：坎(阴) vs 巽(阴) -> 相同（不变）

*   **第三步：得出结论**
    *   只有**上爻变**。
    *   查表可知：上爻变 = **生气**。
    *   **吉凶**：大吉。坎命人住巽方（东南），得生气木（水生木），大吉大利。

### 【案例二】 震卦（☳）为命主，遇到 乾卦（☰）

*   **第一步：画出卦象**
    *   **震卦（主）**：☳ （阴、阴、阳）
    *   **乾卦（客）**：☰ （阳、阳、阳）

*   **第二步：逐爻对比**
    *   **上爻**：震(阴) vs 乾(阳) -> **变**
    *   **中爻**：震(阴) vs 乾(阳) -> **变**
    *   **下爻**：震(阳) vs 乾(阳) -> 不变

*   **第三步：得出结论**
    *   **上爻和中爻变**。
    *   查表可知：上中变 = **五鬼**。
    *   **吉凶**：次凶。震木遇五鬼火（木生火，泄气），主口舌是非。

### 【案例三】 离卦（☲）为命主，遇到 坎卦（☵）

*   **第一步：画出卦象**
    *   **离卦（主）**：☲ （阳、阴、阳）
    *   **坎卦（客）**：☵ （阴、阳、阴）

*   **第二步：逐爻对比**
    *   **上爻**：离(阳) vs 坎(阴) -> **变**
    *   **中爻**：离(阴) vs 坎(阳) -> **变**
    *   **下爻**：离(阳) vs 坎(阴) -> **变**

*   **第三步：得出结论**
    *   **三爻全变**。
    *   查表可知：全变 = **延年**。
    *   **吉凶**：小吉。水火既济，阴阳正配，主婚姻美满。

## 快速记忆歌诀（大游年歌）解析

古书里常用的口诀，其实就是把上面的推导结果背下来。

> **歌诀例：** "乾六天五祸绝延生"

**拆解分析：**

1.  **乾**：代表主卦是**乾卦**。
2.  **六**：遇到**坎卦**（口诀顺序第一位） -> **六煞**。
    *   *验证：乾(☰) vs 坎(☵)，上下变 -> 六煞。对！*
3.  **天**：遇到**艮卦**（口诀顺序第二位） -> **天医**。
    *   *验证：乾(☰) vs 艮(☶)，中下变 -> 天医。对！*
4.  **五**：遇到**震卦** -> **五鬼**。
5.  **祸**：遇到**巽卦** -> **祸害**。
6.  **绝**：遇到**离卦** -> **绝命**。
7.  **延**：遇到**坤卦** -> **延年**。
8.  **生**：遇到**兑卦** -> **生气**。

**注意**：不同派别的口诀顺序可能不同（有的按后天八卦序，有的按先天），但**变爻法则**是永恒不变的真理。建议掌握变爻法则，通过逻辑推导，比死记硬背更准确。

## 应用建议

1.  **不用背全图**：只需记住"上生、中绝、下祸、上中五、上下六、中下天、全延、不变伏"这8个对应关系。
2.  **随手推算**：在手掌上排布八卦，看手指的阴阳变化，立刻就能知道吉凶。
