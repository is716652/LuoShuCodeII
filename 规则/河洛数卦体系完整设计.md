# 河洛数卦体系完整设计

## 一、理论基础

### 1.1 河洛数卦体系组成

河洛数卦体系由四个核心部分组成：

| 图谱 | 作用 | 层级 | 关键问题 |
|------|------|------|----------|
| 1. 元气图 | 定本命卦、先天数 | 体（命格） | "我是谁？" |
| 2. 化工图 | 判断元气强弱（得令否） | 体之强弱 | "我有没有力量？" |
| 3. 反局图 | 排查先天冲突（时空错位） | 体之隐患 | "我有没有硬伤？" |
| 4. 游年图 | 推演方位/流年吉凶 | 用（应事） | "我去哪里/做什么会吉/凶？" |

### 1.2 体用结合原理

- **上文所述的"元气图""化工图""反局图"，解决的是 "你是谁？你的先天命格如何？"（体）**
- **八卦游年图，解决的是 "你在不同方位、不同年份会遇到什么？"（用）**

二者配合，就构成了一个完整的预测模型：  
先看命盘（体）——再看流年/方位吉凶（用）

### 3.2 体用配合公式

```
终极配合公式：
- 吉位 + 强命 = 大吉
- 吉位 + 弱命 = 吉中藏险
- 凶位 + 强命 = 可抗灾
- 凶位 + 弱命 = 大凶之象
```

## 四、用户交互流程

1. **输入阶段**：用户输入出生年月日时
2. **计算阶段**：
   - 通过万年历数据获取干支信息
   - 查询元气图确定本命卦
   - 查询化工图判断得令状态
   - 查询反局图排查风险
   - 查询游年图获取方位吉凶
3. **展示阶段**：
   - 展示基础信息
   - 展示体用分析结果
   - 提供方位应用指南
4. **应用阶段**：指导用户在生活中运用方位吉凶

这套UI设计将复杂的河洛数卦理论转化为直观的可视化界面，让用户能够轻松理解和应用传统命理智慧。
