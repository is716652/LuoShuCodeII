# 化工相反之图

| 卦名 | 五行 | 相反时令/条件 | 终止/持续时间 |
| :---: | :---: | :--- | :--- |
| 乾 | 金 | 丑月 | 十八日 |
| 乾 | 金 | 辰月 | 十八日 |
| 乾 | 金 | 未月 | 十八日 |
| 乾 | 金 | 戌月 | 十八日 |
| 坎 | 水 | 自夏至以后 | 至白露终止 |
| 离 | 火 | 自冬至以后 | 至惊蛰终止 |
| 兑 | 金 | 自春分以后 | 至芒种终止 |
| 震 | 木 | 自秋分以后 | 至大雪终止 |


这组口诀非常关键，它揭示了河洛理数中“卦气反局”与“五行寄生”的深层逻辑。

你可能会发现，这组口诀和上一组（坤土旺十八日、离火夏至等）有很多矛盾之处。比如：上一组说“土旺于四季末18天”，这一组却说“乾金旺于四季末18天”；上一组说“离火旺于夏至后”，这一组却说“坎水旺于夏至后”。

这正是“元气相反之图”（大凶）的理论依据。

我们可以把这组口诀分为两部分来解读：

🧱 第一部分：乾金的“篡位”——四季末十八日

口诀：
乾金丑月十八日
乾金辰月十八日
乾金未月十八日
乾金戌月十八日

解读：
在正统的五行理论中，四季末的18天是“土”当权（坤土旺）。但是，在这张“反局”口诀中，变成了“乾金”当权。

*   为什么是“反”？
    *   正局： 土生金（坤土生乾金），土是金的母亲，母亲爱护孩子，金气顺理成章地生成。
    *   反局： 金旺则土衰。如果在四季末这本该土旺的时候，反而金气太旺，这就叫“母被子泄”或“金多土虚”。
    *   象义： 这是一种“过犹不及”的状态。土代表信用、稳重、脾胃；金代表义气、决断、肃杀。当金在土该旺的时候反客为主，意味着这个人性格过于刚烈、刻薄，缺乏包容心，或者身体上出现“土被金伤”的脾胃问题。

🔄 第二部分：四时卦气的“错乱”

这一部分完全打乱了正常的四季规律，把原本属于某个季节的卦气，强行换到了另一个季节。

💧 坎水：自夏至以后 至白露终止
*   正局： 夏至后是夏天，应该是离火最旺。
*   反局： 这里却说坎水旺。
*   象义： “夏日水泛”。这是典型的水火相冲。夏天火气正炎，却被水强行浇灭。这代表运势上的“背道而驰”，好比在热火朝天的事业中突然遭遇冷水，主官非、破财、心肾疾病。

🔥 离火：自冬至以后 至惊蛰终止
*   正局： 冬至后是冬天，应该是坎水最旺。
*   反局： 这里却说离火旺。
*   象义： “冬日火烧”。冬天水气正寒，却被火强行蒸腾。这代表“燥气攻心”，主性格暴躁、心血系统疾病，或者在该收藏的季节（冬天）因为冲动而破耗元气。

⚔️ 兑金：自春分以后 至芒种终止
*   正局： 春分后是春天，应该是震木最旺（万物生长）。
*   反局： 这里却说兑金旺。
*   象义： “春行秋令”。春天是生发的季节，却出现了秋天的肃杀之气。这会“损伤生机”，好比刚发芽的小草被霜打。主婚姻不顺（兑为少女，被木克）、呼吸系统疾病，或者做事半途而废。

🌲 震木：自秋分以后 至大雪终止
*   正局： 秋分后是秋天，应该是兑金最旺（万物凋零）。
*   反局： 这里却说震木旺。
*   象义： “秋行春令”。秋天是肃杀的季节，却强行要有生机。这代表“逆势而为”，好比在该收割的季节去播种。主劳而无功、筋骨受伤（木被金克），或者性格过于固执，不知变通。

📌 总结：这组口诀的用途

结合你之前问的“元气相反之图-生年得之大凶”，这组口诀就是用来验证“大凶”的工具。

*   正局（上一组口诀）： 讲的是“顺”，顺应天时，所以“大吉”。
*   反局（这一组口诀）： 讲的是“逆”，逆乱天时。
    *   如果你的生年元气是坎水，却落在了夏至后（本该火旺你却水旺），这就是“犯旺”。
    *   如果你的生年元气是乾金，却落在了四季末（本该土旺却金旺），这就是“盗气”。

结论：
这组口诀描述的是一种“气候异常”的状态。在命理学中，如果一个人的出生时间（年、月）符合这种“反局”特征，意味着他/她的先天气场与大自然的节奏是错频的，因此推断为“生年得之大凶”（人生阻力大，多波折）。
